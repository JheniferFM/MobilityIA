
# Nexus Mobility AI

## Descrição do problema
A mobilidade urbana no Distrito Federal enfrenta desafios como pouca previsibilidade nos horários, rotas pouco eficientes, insegurança em alguns trajetos e baixa integração entre transporte público, alternativas sustentáveis e informação em tempo real para os cidadãos.

## Solução proposta
O Nexus Mobility AI é uma plataforma inteligente que reúne IA, dados de mobilidade e interfaces claras para ajudar usuários a planejar viagens, acompanhar lotação, descobrir rotas seguras, acessar informações sobre ônibus e participar de soluções colaborativas como caronas.

## Tecnologias utilizadas
- React 18
- TypeScript
- Vite
- React Router
- Tailwind CSS
- Recharts
- Motion (Framer Motion)
- Lucide React
- Radix UI
- Sonner

## Como instalar
1. Clone o repositório.
2. Acesse a pasta do projeto.
3. Execute:
   ```bash
   npm install
   ```

## Como executar
Após a instalação, execute:
```bash
npm run dev
```
O projeto ficará disponível no navegador no endereço padrão do Vite.
